# 阶段二 · 零基础上手


## 这个阶段讲什么

读完阶段一，你知道 AI 是什么了——但光知道没用，得**用起来**才算数。

这个阶段的核心是把市面上最常用的 AI 工具让你可以直接上手用起来。

不追求大而全的工具介绍，只聚焦**最新、最实用**的 AI 上手与使用。
