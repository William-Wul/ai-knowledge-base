# 学新不学旧，用实不用虚


如果让我总结一下小白学习 AI、使用 AI 的原则，那我觉得就是这八个字——**"学新不学旧"和"用实不用虚"**。

## 一、学习 AI 的原则：学新不学旧

为什么？因为 AI 一直在更新而且速度很快，并且方向一直是门槛越来越低、效果越来越好。

这也就意味着，你原来花了大功夫解决的上手门槛问题，可能很快就没有了；你原来努力了半天能达到的最好学习效果，结果一个新模型出来就被碾压了。

所以尽量去学一些新的东西，哪怕用不上，也要先去了解——这样可以帮你更有先见之明地理清楚 AI 学习和使用的思路，甚至帮你做好该做的职场应对。

举个例子，比如 Openclaw（小龙虾），说白了就是给大模型装上了一个框架，这个框架权限足够大，可以操作一些你的电脑应用和后台，理论上能帮你直接干活，于是最近几个月引爆了一波付费装小龙虾、学怎么使用的热潮。

![Openclaw 相关截图](/images/stage-2/WPFsbPJ1to7XxAx5PuEcPuQJnSf.png)

但如果你现在去学，我觉得完全没必要，因为这玩意儿已经被各种新东西降低门槛了，普通人直接用就好了，没有必要再去学了。比如智谱清言的 Autoclaw（澳龙）、腾讯的 Qclaw、字节的飞书 aily，甚至连有道都有龙虾产品，直接开用就可以了。

你可能会说，但是这不是装在我电脑上的，而且不会被割韭菜、更贵吗？

首先工作电脑是不能装龙虾的。然后你的个人电脑装龙虾的代价可能比你被各家平台收的积分费代价更大。最后即使你折腾上了自己电脑的龙虾，中间配 API、调各家模型也是要付费的——我第一次使用因为不了解用量和配置，直接调用了 Claude 的模型，一上午就烧掉了几十美元，这个学习成本完全没必要。本地装龙虾和直接用云部署的龙虾，对普通人而言没啥区别，尤其是你还没想清楚要拿龙虾做什么的时候。

当然如果你学会了用更新的 Codex、Claude Code，包括国内的 Coze、Workbuddy、秒哒之类的，你会发现龙虾这玩意儿其实挺鸡肋的。

这就是学新不学旧——现在各大厂家激烈竞争，让他们卷去吧，我们要学就学最新的东西。

再比如生图，如果你还在漫天收集提示词，不如去试试新出的 GPT-image-2，国内也有平台接入了这个模型。你会发现提示词根本不重要，大模型够不够牛才是最关键的。

## 二、使用 AI 的原则：用实不用虚

好，假设你已经学习到了最新的 AI 知识，知道了 Claude Code 是目前小白用起来最好用的 vibecoding 工具，知道了 ChatGPT 是目前地表最强生图工具，然后你就开始花大价钱去开通 Claude 的订阅，还要想办法解决魔法上网和网络纯净度的问题，还要额外充一个 ChatGPT 的订阅——还没开始用上呢，就先花出去 1000+ 软妹币。

![先买工具后茫然的状态](/images/stage-2/MswYbfdD5ozp7Wxg2XNcIID2nad.png)

然后呢？

你真的知道自己用 Claude Code 该做什么吗？你用 ChatGPT 的生图模型做了几张牛逼 plus 的梗图之后，还能想到做些什么吗？

如果你知道，那恭喜你，你解锁了强大的生产力工具。

但如果你不知道，同时又是会计算成本的人，那建议你不要盲目去冲最新的 AI 工具。

比如你只是想日常工作提效，我觉得用好豆包就够了——改文案、做 PPT、语音转文字、生图、生视频、截图提问、翻译……这些功能你用好了，可能不是 100 分的效果，但 80 分+的效果总归还是不错的，最主要是没有门槛、几乎也没有成本。这就是"实"。花了 1000 多软妹币，最后拔剑四顾心茫然，这就是"虚"。

**先想好需要解决什么问题，再想需要什么类型的工具来达到什么效果，然后才是选择具体使用的工具。** 而不是反过来，先把最牛的工具给自己装备上，再思考拿这个工具干什么。尤其是当下 AI 的费用消耗其实不低，Token 贩子都赚翻了，咱们就别上赶着去当韭菜了。
