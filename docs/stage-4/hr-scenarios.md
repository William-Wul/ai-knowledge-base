# 人资场景

> 本文预计阅读时间：15 分钟 &nbsp;·&nbsp; <span class="stage-badge stage-3">阶段三 · 场景深耕</span>

---

HR 的工作有大量需要"动脑写字"的环节——写 JD、写通知、设计面试问题、起草政策……这些正好是 AI 最擅长的事。这篇文章整理了 HR 最常用的 8 个 AI 使用场景，每个都有可以直接复制的提示词模板。

---

## 场景一：写职位描述（JD）

传统 JD 写起来又费时又模板化。用 AI，5 分钟出一个初稿。

**模板：**
```
请帮我写一份职位描述（JD），要求如下：
- 职位名称：【例如：高级财务分析师】
- 所在部门：【例如：财务部】
- 核心职责：【例如：负责月度财务报告、预算管理、成本分析】
- 任职要求：【例如：本科及以上，财务/会计相关专业，3年以上经验，熟悉 Excel】
- 公司风格：【例如：互联网公司，文化比较年轻化】
- 语言风格：简洁有吸引力，不要官腔
```

**使用提示：** 生成后检查职责描述是否符合实际岗位要求，调整具体细节。

## 场景二：设计面试问题

针对不同岗位设计有质量的面试问题，尤其是行为面试题（STAR 结构）。

**模板：**
```
我需要面试一名【职位名称】，请帮我设计 8 个面试问题，要求：
- 包含 3 个专业能力题：考察【核心技能，例如：数据分析能力】
- 包含 3 个行为面试题：考察【例如：抗压能力、团队协作】
- 包含 2 个动机题：了解候选人的职业规划和对岗位的理解
- 每个问题附上"理想回答的关键点"，方便我打分
```

## 场景三：起草 offer 信和录用通知

**模板：**
```
请帮我起草一封录用通知邮件，信息如下：
- 候选人姓名：【用"候选人"代替，不要填真名】
- 应聘岗位：【例如：市场专员】
- 入职日期：【例如：下周一】
- 需要携带材料：【例如：身份证原件、毕业证、离职证明】
- 语气：正式、热情，让候选人感受到被重视
- 字数：200 字左右
```

## 场景四：设计培训课程大纲

**模板：**
```
请帮我设计一个内部培训课程大纲，要求：
- 培训主题：【例如：新员工入职培训 / AI 工具使用培训】
- 目标学员：【例如：全体新入职员工，无技术背景】
- 培训时长：【例如：半天，约 4 小时】
- 核心目标：【例如：了解公司文化、熟悉工作流程、掌握基础工具使用】
- 请给出：模块划分、每模块时长、核心内容要点、推荐的授课形式
```

## 场景五：起草员工通知公告

**模板：**
```
请帮我写一则内部通知，内容如下：
- 通知类型：【例如：关于调整考勤制度的通知】
- 主要变化：【例如：从下月起，弹性上班时间改为 9:00-10:00 可选择打卡，核心工时 10:00-17:00 不变】
- 生效日期：【例如：5 月 1 日】
- 语气：正式，但不要太生硬
- 格式：标准公文格式，包含标题、正文、落款
```

## 场景六：处理员工关系的沟通话术

遇到需要跟员工谈敏感话题（绩效面谈、违规处理、离职沟通），提前用 AI 准备沟通思路。

**模板：**
```
我需要跟一名员工进行一次谈话，情况如下：
- 谈话背景：【例如：该员工连续三个月绩效未达标，本次是第一次正式谈话】
- 我的目标：【例如：了解原因，帮助员工改善，同时明确说明若不改善的后果】
- 员工可能的反应：【例如：可能会情绪化，或者觉得考核不公平】

请帮我：
1. 列出谈话的建议流程（分步骤）
2. 给出开场白的参考话术
3. 提示我需要注意的沟通细节
```

## 场景七：制定绩效考核指标

**模板：**
```
请帮我为【职位名称】设计一套绩效考核指标（KPI），要求：
- 工作性质：【例如：该岗位主要负责客户服务，包含售前咨询和售后处理】
- 考核周期：【季度 / 月度】
- 请给出 5-8 个核心指标，每个指标包含：指标名称、衡量方式、权重建议
- 风格：量化为主，尽量有数据支撑，避免太主观的描述
```

---

## 场景八：写离职面谈问题清单

**模板：**
```
请帮我设计一份离职面谈问题清单，目的是：
- 了解员工真实离职原因
- 收集对公司的改进建议
- 在员工心情好的时候维护雇主品牌

请设计 10 个问题，语气要轻松，让员工愿意说真心话，而不是走过场。
```

---

## 一句话总结

HR 工作中 **70% 的文字工作** 都可以用 AI 做初稿，你来审核和调整。时间省出来，用在真正需要人的地方：判断、沟通、关系维护。
