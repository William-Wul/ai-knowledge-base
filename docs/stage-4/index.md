# 阶段四 · 工作场景实战

> 本阶段按岗位分章，挑和你相关的读即可

## 这个阶段讲什么

到了这里，你已经知道"AI 是什么"、"怎么用"、"Agentic 是什么意思"——但所有这些都要**落到你具体岗位的日常工作**里，才算真的掌握了。

这个阶段不是概念课，是**岗位实战手册**。每一篇都按"这个岗位最常做的事 → AI 在哪些环节能帮上 → 怎么帮 → 踩过的坑"的结构来写，直接给你流程、提示词、模板。

不需要通读。**找你岗位的那一篇**读完，对着套就行。

## 本阶段文章（按岗位）

- [产品经理 · AI 实战](/stage-4/pm-scenarios) — 竞品分析、用户调研、PRD、数据分析、方案策划的 AI 工作流（最深入的一篇）
- [人资场景](/stage-4/hr-scenarios) — JD 撰写、简历筛选、面试问题设计、培训材料生成
- [财务场景](/stage-4/finance-scenarios) — 报表解读、费用分析、财务问答机器人搭建
- [法务场景](/stage-4/legal-scenarios) — 合同审阅、条款比对、合规检查
- [运营 / 产品场景](/stage-4/ops-product-scenarios) — 用户反馈分类、活动文案、数据仪表板解读
- [创意 / 内容场景](/stage-4/creative-scenarios) — 文案、海报、视频脚本、图片生成
