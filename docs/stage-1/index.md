# 阶段一 · 快速认知

> 本阶段预计阅读时间：20 分钟

## 这个阶段讲什么

ChatGPT、DeepSeek、豆包、文生图、Agent、多模态……这些名词你多半听过，但你大概率说不清楚**它们之间的关系**。

这个阶段不教任何"具体怎么用 AI"，只解决两件事：

1. **AI 到底是什么**——从人类大脑一路讲到大模型，用一根主线把来龙去脉串清楚
2. **那些 AI 圈天天在说的词，到底是什么意思**——术语扫盲，让你以后听同事/新闻讲 AI，不再一脸懵

读完你会得到一个**完整的 AI 世界观**。后面所有的学习都建立在这个世界观之上。
