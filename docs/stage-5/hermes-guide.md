# Hermes 配置与使用

> 预计阅读时间：8 分钟 · 适合想搭建个人AI工作流的用户

Hermes 是一款 AI Agent 网关工具，可以把 AI 大模型连接到飞书、微信等通讯平台，让你通过日常聊天的方式指挥 AI 执行各种任务。

---

## 一、Hermes 能做什么？

**核心功能：** 把 AI 变成你飞书/微信里的一个"智能助手"。

你可以在飞书里给它发消息：
- "帮我搜索一下最近的AI行业动态"
- "用XX模板帮我写一篇小红书文案"
- "提醒我明天下午3点开会"

Hermes 收到消息后，调用 AI 模型处理，然后把结果发回给你。整个过程就像跟一个真人助手聊天一样自然。

---

## 二、核心概念

**Hermes Gateway：** 后台服务，负责接收和转发消息
**Skills（技能）：** 可以给 Hermes 装载不同的"技能"，比如搜索技能、写作技能、数据分析技能。装了什么技能，它就会什么本事
**Feishu Bot（飞书机器人）：** 在飞书里创建一个机器人，连接到 Hermes，就能在飞书里跟 AI 对话

---

## 三、安装配置概要

Hermes 的安装需要一定的技术操作（终端命令行），建议在 Claude Code 的辅助下完成。大致流程：

1. **安装 Hermes：** 在终端执行安装命令
2. **配置模型：** 在配置文件中设置 AI 模型。推荐使用 MiniMax M2.7（国内API `api.minimaxi.com`，延迟低、稳定）
3. **连接飞书：** 在飞书开放平台创建一个机器人应用，获取凭证后配置到 Hermes
4. **安装技能：** 把需要的 Skill 文件放到 Hermes 的 skills 目录下
5. **启动服务：** 运行 `hermes gateway start`

---

## 四、常用技能推荐

| 技能 | 功能 | 说明 |
|------|------|------|
| baidu-search | 百度搜索 | 让 Hermes 能联网搜索信息 |
| fanren_xiaohongshu_style | 小红书风格写作 | 按特定风格模板生成内容 |
| 自定义 Skill | 任意功能 | 你可以自己写 Skill 模板 |

---

## 五、使用建议

1. **Hermes 适合做轻量级、模板化的任务**——比如按固定模板写文案、定时发送消息、简单的信息查询
2. **不适合做复杂的多步骤工程任务**——比如搭建网站、安装软件包。这些交给 Claude Code 更合适
3. **M2.7 模型的指令跟随能力有限**——给它的指令越简单、越具体，效果越好。复杂指令容易跑偏
4. **用 hard-template Skill 来补偿模型能力**——把你想要的输出格式写死在 Skill 模板里，模型只需要填内容

## 六、与其他工具的分工

| 工具 | 适合做的事 |
|------|-----------|
| Hermes/银月 | 日常对话、知识库查询、模板化内容生成、定时任务 |
| Claude Code | 写代码、建项目、文件操作、复杂工程任务 |
| Claude.ai | 方案规划、深度分析、内容润色、长文写作 |

**三者配合使用效果最好**——Claude.ai 想方案，Claude Code 执行，Hermes 做日常运营。
