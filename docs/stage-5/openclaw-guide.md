# OpenClaw 配置与使用

> 预计阅读时间：8 分钟 · 适合想体验AI自主操作电脑的用户

OpenClaw（小龙虾）是一款开源的 AI Agent 工具，它能直接操作你的电脑——打开应用、浏览网页、填写表格、整理文件。在 GitHub 上几周内就获得了数万颗星，是目前最热门的个人 AI Agent 工具之一。

---

## 一、OpenClaw 能做什么？

简单说：**你告诉它一个目标，它自己操作电脑帮你完成。**

比如：
- "帮我搜索最近的AI行业报告，下载前三篇"
- "打开我的邮箱，把未读邮件的主题整理成一个列表"
- "帮我在飞书里创建一个新文档，把这段内容填进去"

它不是回答问题，而是**真的在操作你的电脑屏幕**，像一个远程助手一样。

---

## 二、安装步骤

**前提：** Mac 电脑（Windows 支持有限）

1. 在终端执行安装命令（具体命令请查看 OpenClaw 官方 GitHub 页面的最新说明）
2. 安装完成后运行 `openclaw onboard --install-daemon` 初始化
3. 配置 AI 模型——OpenClaw 需要连接一个大模型作为"大脑"。推荐使用 MiniMax M2.7（国内延迟低）或通过 OpenRouter 接入 Claude
4. 配置完成后，OpenClaw 会以后台服务的形式运行

---

## 三、基本使用

启动后，你可以通过飞书、微信或其他已连接的平台给 OpenClaw 发消息，它会在你的电脑上执行操作。

**使用技巧：**

- 指令越具体，执行越准确。"帮我搜AI报告"不如"帮我在Google上搜索'2026年中国AI行业报告'，打开前三个结果"
- 复杂任务拆成小步骤给它，不要一口气让它做太多
- 重要文件操作前先备份

---

## 四、注意事项

1. **安全风险：** 国家网络与信息安全信息通报中心指出 OpenClaw 存在安全风险。使用时务必谨慎，不要让它操作涉及银行、支付等敏感应用
2. **稳定性：** 作为新兴工具，OpenClaw 有时会执行出错或卡住，需要你手动干预
3. **成本：** OpenClaw 本身免费，但调用的大模型 API 需要付费
