# Claude Code 使用指南

> 预计阅读时间：10 分钟 · 适合想了解AI编程工具的读者

Claude Code 是 Anthropic 出品的一款 AI 编程助手，运行在电脑的终端（命令行）里。它不只是帮你写几行代码——你给它一个任务，它能自己写代码、创建文件、执行命令、调试错误，整个过程自主完成。

听起来很像程序员才用的工具？不完全是。本文作者（也就是这个知识库的创建者）是一个完全不懂代码的人，但用 Claude Code 一天之内搭建了这整个知识库网站——从零开始，包括项目初始化、内容结构、密码保护、GitHub 部署、甚至后台管理系统。

**这就是 Claude Code 的厉害之处：你不需要会写代码，你只需要会"说"你想要什么。**

---

## 一、Claude Code 能做什么？

简单来说，Claude Code 能操作你电脑上的终端，帮你完成各种技术任务：

**常见用途：**

- 搭建网站、Web 应用
- 写脚本自动化处理文件（批量重命名、数据整理等）
- 读代码、改代码、修 bug
- 安装和配置各种开发工具
- 操作 Git 进行版本管理和部署
- 处理 API 接口调用

**一个真实案例：本知识库的诞生**

这个 AI 学习知识库网站，就是用 Claude Code 搭建的。全过程大致如下：

1. 告诉 Claude Code："帮我搭一个 VitePress 知识库网站"
2. 它自动执行了：创建项目 → 安装依赖 → 配置导航和侧边栏 → 创建文章目录 → 添加密码保护 → 配置 GitHub Actions 自动部署
3. 最终产出：一个可以在线访问的完整网站

整个过程中，操作者不需要写任何代码，只需要描述需求和确认每一步的操作。

## 二、怎么安装？

**前提条件：**
- 一台 Mac 或 Linux 电脑（Windows 需要用 WSL）
- 已安装 Node.js

**安装步骤：**

在终端里执行一条命令：

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

安装完成后，输入 `claude` 就能启动。

**关于费用：**

Claude Code 本身免费，但它调用的 AI 模型需要付费。有两种方式：

1. **Anthropic 官方 API**——需要国际信用卡充值
2. **通过 OpenRouter 接入**——支持更多支付方式，国内用户更方便。在 openrouter.ai 注册、充值、获取 API Key，然后在终端配置环境变量即可

通过 OpenRouter 接入时，默认使用 Claude Sonnet 4.6 模型，输入 3 美元/百万 token，输出 15 美元/百万 token。日常轻度使用（比如替换一篇文章然后发布），一次任务大概几分钱到一两毛钱。

## 三、基本使用方式

**启动：**

在终端里进入你想工作的目录，然后输入：

```bash
cd ~/你的项目目录
claude
```

你会看到一个对话界面，直接用中文输入你的需求就行。

**常用命令：**

| 命令 | 作用 |
|------|------|
| `/status` | 查看当前连接状态和模型信息 |
| `/model` | 切换模型（比如从 Sonnet 切到 Opus） |
| `/help` | 查看所有可用命令 |
| `Ctrl + C` | 退出当前对话 |

**跟 Claude Code 说话的技巧：**

和在 AI 对话工具里一样，描述越清楚结果越好。但 Claude Code 有一个额外的好处：它能看到你电脑上的文件，所以你可以直接说"打开那个文件""修改第三行"这样的指令。

## 四、省钱技巧

Claude Code 通过 API 计费，用多少扣多少。以下技巧能帮你控制花费：

**1. 一个任务一个对话，做完就关**

对话越长，每轮携带的上下文越多，token 消耗越大。完成一个任务后按 `Ctrl + C` 退出，开始新任务时重新输入 `claude` 开新对话。

**2. 默认用 Sonnet，别一直挂 Opus**

Sonnet 是默认模型，日常任务完全够用。Opus 的价格是 Sonnet 的约 1.7 倍，只在特别复杂的架构设计或调试场景下才值得切换。

**3. 简单操作自己做**

文件复制、重命名这种简单操作，自己在终端敲命令比让 Claude Code 做更快也更省钱。把 Claude Code 留给真正需要 AI 判断的任务。

**4. 用文字描述问题，少发截图**

截图会消耗大量视觉 token。能用文字描述清楚的问题就不要发截图。

## 五、适合什么人？

**程序员**——Claude Code 是目前效率最高的 AI 编程助手之一，能大幅提升开发速度。

**想用 AI 搭建东西的非技术人员**——就像本知识库的作者一样，不懂代码也能用 Claude Code 搭网站、写脚本、做自动化。关键是你要能清楚地描述你想要什么。

**技术管理者**——了解 Claude Code 的能力边界，有助于判断哪些技术任务可以用 AI 加速。

---

## 六、使用时的注意事项

**1. 一定要审核它的操作**

Claude Code 会在你的电脑上执行真实的命令。虽然重要操作会先征求你的确认，但你还是要看清楚它要做什么再同意。特别是涉及删除文件、修改配置等操作。

**2. 重要文件提前备份**

在让 Claude Code 改动重要项目之前，确保你有备份。Git 版本控制是天然的备份机制——每次改动都会记录，可以随时回退。

**3. 别在一个超长对话里做所有事**

长对话不仅消耗更多 token，还会导致 AI 的注意力分散。一个任务一个对话是最佳实践。
