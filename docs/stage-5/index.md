# 阶段五 · AI Agent 使用


**建设中，敬请期待**

## 这个阶段讲什么

前四个阶段，你和 AI 的关系是"你问一句、它答一句"的对话。

到了 Agent 阶段，关系变了：**你下一个任务，AI 自己规划、执行、调用工具、回报结果**。

举例：
- 对话式 AI：你让它"帮我查一下上周周报里出现频率最高的三个关键词"，它只能靠你把周报粘贴给它
- Agent：你说"看一下我的周报文件夹，找出上周出现频率最高的三个关键词，生成一份对比"——它自己去读文件、分析、写报告
