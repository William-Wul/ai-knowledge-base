# AI 热点精选

> 帮你每天花 3 分钟就能看完 AI 圈正在发生什么,不用自己刷十几个平台。

---

## 这个专区是什么

AI 行业信息密度高,单看一家媒体容易漏,全刷又累。

这个专区每天自动从 **11 个国内热榜 + 海外科技圈信息源** 抓取,筛出跟 AI 相关的条目,**按跨平台热度排序 + 由 LLM 自动生成 200-300 字概述**,让你一条一条扫下来就能掌握当日 AI 圈全貌。

每日一期,GitHub Actions 定时更新。

---

## 覆盖平台

**国内(11 个,抓取热榜排名)**

微博热搜 · 知乎热榜 · 今日头条 · 百度热搜 · 抖音热点 · B 站热门 · 百度贴吧 · 澎湃新闻 · 华尔街见闻 · 财联社 · 凤凰网

**海外(通过 RSS 接入)**

Hacker News(科技圈讨论热帖)· GitHub Trending(当日热门开源项目)

> **关于 X / Twitter**:原本考虑加入,但该平台无稳定公开热榜数据源,且 AI 话题在 X 上的命中率不如 Hacker News + GitHub Trending 精准。当前版本暂不接入,如后续确认需要会通过付费 API 单独加。

## 热度等级怎么定

每条热点的 🔥 数量(1-5)是 LLM 综合四个维度打分后的结果:

- **重要性** — 事件在 AI 行业的战略级别(头部公司旗舰发布 / 重大并购 / 监管政策 > 二线产品更新)
- **时效性** — 今日首发硬核事件 > 持续讨论中的老话题
- **讨论度** — 跨平台覆盖面 + 各平台上的排名位置(越靠前越热)
- **影响面** — 对普通用户或工作者是否有直接影响

**火焰数的直觉含义:**

| 火焰 | 直觉含义 |
| --- | --- |
| 🔥🔥🔥🔥🔥 | 全网级大事(旗舰发布 / 重大并购 / 能力质变) |
| 🔥🔥🔥🔥 | 重要行业进展(头部公司更新 / 开源 SOTA / 重要政策) |
| 🔥🔥🔥 | 值得关注(细分赛道突破 / 专业圈讨论) |
| 🔥🔥 | 有点意思(单平台热议 / 工具小更新) |
| 🔥 | 边缘命中(顺带提到 AI 的财经、娱乐新闻) |

---

## 每条热点包含什么

每条折叠卡片默认显示 **事件标题 + 综合热度分**,点开展开后可看到:

- **200-300 字概述** — LLM 自动生成,讲清楚"发生了什么 + 为什么重要 + 对你有什么影响"
- **代表链接** — 跳转到跨平台排名最靠前的那条原帖
- **命中关键词** — 被关键词表里的哪个词抓到的

## 查看日报

所有日报按日期列在**左侧边栏**(点击对应日期直接跳转)。每天北京时间 21:00 自动更新当日报告。

---

## 更新频率

每天一次,北京时间 21:00 左右自动产出。遇到当日 AI 新闻极少或抓取失败时,当日文件可能缺失(不覆盖次日日报)。

---

## 关键词表怎么维护

关键词表(`frequency_words.txt`)维护在仓库里,持续迭代。当前分三层:

- **核心层**:AI + 模型、大模型、LLM、AGI、生成式
- **玩家层**:OpenAI、ChatGPT、Claude、Gemini、DeepSeek、豆包、通义、文心、Kimi、Llama 等
- **应用层**:AI Agent、AI 编程、Cursor、MCP、AI 视频、AI 绘画、具身智能、人形机器人、AI 眼镜 等

如果你发现某类 AI 话题长期漏抓,或某个词噪音太大,告诉我调整即可。

---

## 注意事项

- 这个专区是**机器自动产出**,不做人工解读。想看深度解析请到 [📰 AI 新闻](/news/) 看周报
- 概述由 LLM 生成,可能存在事实偏差,重要信息请以代表链接里的原文为准
- 抓取时间为每日早晚各一次(北京时间约 08:00 和 20:00)
