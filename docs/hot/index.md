---

## 这个专区是什么

AI 行业信息密度高,单看一家媒体容易漏,全刷又累。

这个专区每天自动从国内外十几个热榜和科技社区抓取,筛出跟 AI 相关的条目,按跨平台热度排序 + 由 LLM 自动生成 200-300 字概述,让你一条一条扫下来就能掌握当日 AI 圈全貌。

---

## 热度等级怎么定

每条热点的 🔥 数量(1-5)是综合四个维度打分后的结果:

- **重要性** — 事件在 AI 行业的战略级别(头部公司旗舰发布 / 重大并购 / 监管政策 > 二线产品更新)
- **时效性** — 今日首发硬核事件 > 持续讨论中的老话题
- **讨论度** — 跨平台覆盖面 + 各平台上的排名位置(越靠前越热)
- **影响面** — 对普通用户或工作者是否有直接影响

**火焰数的直觉含义:**

| 火焰 | 直觉含义 |
| --- | --- |
| 🔥🔥🔥🔥🔥 | 全网级大事(旗舰发布 / 重大并购 / 能力质变) |
| 🔥🔥🔥🔥 | 重要行业进展(头部公司更新 / 开源 SOTA / 重要政策) |
| 🔥🔥🔥 | 值得关注(细分赛道突破 / 专业圈讨论) |
| 🔥🔥 | 有点意思(单平台热议 / 工具小更新) |
| 🔥 | 边缘命中(顺带提到 AI 的财经、娱乐新闻) |
