# AI 编程

> 这个专区面向有编程经验的技术人员，内容涉及 AI 编程工具、开发实践和技术深度话题。

---

## 专区定位

如果说其他阶段的内容是"用 AI 工具提效"，那这个专区是"用 AI 改变编程方式"。

这里的内容假设你有一定的编程基础——至少熟悉一种编程语言、用过终端命令行、了解基本的开发流程。如果你完全没有编程经验，建议先从阶段五的 [Claude Code 使用指南](/stage-5/claude-code-guide) 开始了解。

---

## 计划覆盖的话题

**AI 编程工具深度对比** — Cursor、Claude Code、GitHub Copilot、通义灵码等主流工具的优劣势和适用场景。

**Vibe Coding 实践** — 用自然语言描述需求，让 AI 生成完整代码的工作方式。从理念到实际项目的全流程。

**AI Agent 开发** — 如何用 LangChain、n8n 等框架搭建自己的 AI Agent。

**本地大模型部署** — 在自己的电脑上跑开源大模型（Ollama + DeepSeek / LLaMA），不依赖云端 API。

**Prompt Engineering 进阶** — 面向开发者的提示词工程，包括系统提示词设计、Few-shot Learning、Chain-of-Thought 等技术。
