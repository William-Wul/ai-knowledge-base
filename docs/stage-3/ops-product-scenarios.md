# 运营 / 产品场景

> 本文预计阅读时间：15 分钟 &nbsp;·&nbsp; <span class="stage-badge stage-3">阶段三 · 场景深耕</span>

---

运营和产品的日常工作里，有大量需要"想内容、写文字、分析信息"的环节。这些恰好是 AI 发挥最大价值的地方。这篇文章整理了 8 个最实用的场景。

---

## 场景一：生成推文 / 公众号文章初稿

**模板：**
```
请帮我写一篇【平台，例如：微信公众号 / 微博 / 小红书】的内容，要求：
- 主题：【例如：我们公司新上线的 XX 功能介绍】
- 目标受众：【例如：25-35 岁的职场白领】
- 内容目标：【例如：让用户了解新功能，引导他们试用】
- 字数：【例如：600-800 字】
- 风格：【例如：轻松有趣，不要太广告感 / 专业严肃 / 有干货的实用风格】
- 结尾带一个行动引导：【例如：引导点击链接 / 评论区互动 / 转发分享】
```

## 场景二：活动策划方案初稿

**模板：**
```
请帮我起草一个活动策划方案框架，内容如下：
- 活动类型：【例如：线上用户增长活动 / 线下品牌体验活动 / 内部员工团建】
- 核心目标：【例如：拉新 XX 人 / 提升品牌曝光 / 增强团队凝聚力】
- 时间范围：【例如：一个月】
- 预算级别：【例如：低预算 / 中等预算，不需要给具体数字】
- 目标人群：【例如：现有用户 / 潜在用户 / 公司内部员工】

请给出：活动主题方向（3 个备选）、活动形式、执行步骤、关键指标
```

## 场景三：用户调研问卷设计

**模板：**
```
请帮我设计一份用户调研问卷，调研目的是：
【例如：了解用户对现有 APP 的满意度，以及对新功能的期望】

要求：
- 问题数量：10-12 题，控制在 3 分钟内填完
- 包含：选择题（单选/多选）+ 评分题 + 1-2 道开放问答
- 避免引导性问题
- 问卷结构：开场说明 → 使用习惯 → 满意度评估 → 改进需求 → 开放反馈
```

## 场景四：竞品分析报告结构

**模板：**
```
我需要做一份竞品分析，请帮我设计分析框架和提纲：
- 我方产品类型：【例如：B2B SaaS 工具 / 消费品 / 线下服务】
- 主要竞品数量：【例如：3 个】
- 分析目的：【例如：找出差异化优势，为下季度产品规划提供依据】

请给出：
1. 分析维度建议（哪些方面值得对比）
2. 报告结构和各部分内容提纲
3. 每个维度建议收集哪些信息
```

## 场景五：写产品需求说明

产品同学把功能需求描述清楚，让研发能看懂，一直是个费时的工作。

**模板：**
```
请帮我写一个产品功能需求说明，信息如下：
- 功能名称：【例如：用户消息免打扰设置】
- 背景和目的：【例如：用户反馈收到太多通知打扰，需要让用户自定义通知偏好】
- 核心用户场景：【例如：用户在睡觉前想关闭所有推送，次日早上自动恢复】
- 预期功能点：【例如：可设置免打扰时间段、可选择哪些类型通知免打扰、可设置例外（如紧急联系人不免打扰）】

请整理成：功能描述 + 用户故事（User Story）格式 + 注意事项和边界条件
```

## 场景六：数据报告的文字解读

你有一份数据，需要写成汇报材料让领导看懂。

**模板：**
```
请帮我将以下数据解读成一段清晰的文字分析（数据已脱敏，用比例代替绝对值）：

数据情况：
- 本月 DAU（日活）环比上月增长 X%，同比去年同期增长 Y%
- 新用户占比 Z%，留存率（次日/7日/30日）分别为 A%、B%、C%
- 核心功能使用率：功能一 M%，功能二 N%，功能三 P%
- 用户投诉最多的 TOP3 问题：【问题描述】

请写一段 300 字左右的数据解读，适合向业务负责人汇报，重点突出亮点和问题，语气客观。
```

## 场景七：起草用户反馈回复话术

客服或运营经常需要回复用户投诉和反馈，AI 帮你统一语气和表达。

**模板：**
```
请帮我写一段用户投诉的回复话术，情况如下：
- 投诉类型：【例如：用户反映 App 经常崩溃，数据丢失】
- 当前处理状态：【例如：技术团队已确认问题，预计 3 个工作日修复】
- 能否给出补偿：【例如：可以赠送 7 天会员】
- 语气要求：诚恳道歉，给出明确解决时间，不说空话

请给出：标准回复模板（可以批量用）+ 个性化版本（针对长期用户的更真诚版本）
```

## 场景八：头脑风暴——想不到好创意时

**模板：**
```
我需要围绕【主题，例如：双十一促销活动】做创意头脑风暴，背景是：
- 产品/服务：【简单描述】
- 目标用户：【描述】
- 想达到的效果：【例如：话题性强，适合社交传播】
- 现有的限制：【例如：不能涉及打折，预算有限，必须是线上活动】

请给我 10 个不同方向的创意，每个创意 2-3 句话描述，要有新意，不要老套路。
```

## 运营 / 产品使用 AI 的最大价值

运营和产品工作的核心是**判断和创意**——这两件事 AI 给不了你。但它能帮你把"有了方向后的执行部分"加速 3-5 倍。

把 AI 当成你的"文字执行助理"：你来定方向，它来帮你把想法变成文字初稿。
