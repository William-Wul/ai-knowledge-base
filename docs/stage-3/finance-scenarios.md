# 财务场景

> 本文预计阅读时间：12 分钟 &nbsp;·&nbsp; <span class="stage-badge stage-3">阶段三 · 场景深耕</span>

---

财务工作的核心是数字和准确性，AI 不能替你算数、不能保证数字对，但它非常擅长**把数字背后的故事用文字讲清楚**。这恰好是财务人员最费时的那类工作。

> **重要提醒**：使用 AI 处理财务内容时，只输入脱敏数据（用假数字替代真实数字），任何真实财务报表、实际收入利润数据都不要输入 AI。

---

## 场景一：撰写财务分析报告文字部分

你已经有数据了，让 AI 帮你把分析结论写成流畅的文字。

**模板：**
```
请帮我撰写一段财务分析报告的文字说明，内容如下：

数据背景（已脱敏）：
- 本季度营收同比增长 X%，较上季度环比下降 Y%
- 主要增长来自【业务线/产品类别】，贡献占比 Z%
- 运营成本上升 W%，主要原因是【人力/原材料/市场推广】
- 净利率为 M%，低于/高于年度目标 N 个百分点

请帮我：
1. 写一段 200 字左右的整体表现总结
2. 分析增长和下滑的主要原因
3. 提出 2-3 个可以关注的改善方向
语气：专业、客观，适合向管理层汇报
```

## 场景二：写费用说明和审批备注

报销或付款审批时，经常需要写一段说明。AI 帮你把事情说清楚。

**模板：**
```
请帮我写一段费用说明，用于审批备注：
- 费用类型：【例如：差旅费 / 市场推广费 / 咨询服务费】
- 业务背景：【例如：出差至北京参加行业峰会，代表公司与潜在合作方洽谈】
- 费用构成：【例如：机票+住宿+餐饮，总计约 XXXX 元（用假数字）】
- 预期收益/必要性：【例如：有助于推进 XX 项目合作谈判】
- 字数：150 字以内，语言简洁有说服力
```

## 场景三：解读财务政策和会计准则

遇到不熟悉的准则或政策，让 AI 用大白话解释。

**模板：**
```
请帮我用简单易懂的语言解释以下内容，假设我是一名非财务背景的管理者：

【把你想了解的政策条款、准则名称，或者一段你看不懂的文件粘贴到这里】

请解释：
1. 这条规定的核心意思是什么
2. 对我们公司的实际操作有什么影响
3. 有什么需要特别注意的地方
```

## 场景四：制作汇报 PPT 的内容框架

向管理层或董事会汇报时，先让 AI 帮你搭框架。

**模板：**
```
我需要准备一个财务季度汇报，请帮我设计 PPT 的内容框架：
- 汇报对象：【例如：公司高管层 / 董事会】
- 汇报时长：【例如：20 分钟】
- 核心主题：【例如：Q1 财务表现回顾 + Q2 预算规划】
- 这次汇报想重点传达的信息：【例如：成本控制成效显著，但营收增速需要关注】

请给出：幻灯片页数建议、每页的标题和核心内容要点
```

## 场景五：起草财务相关的内部邮件

**模板：**
```
请帮我写一封财务相关的内部邮件：
- 邮件目的：【例如：提醒各部门在本月 25 日前完成费用报销申请，逾期不予受理】
- 收件人：【例如：各部门负责人】
- 需要说明的注意事项：【例如：金额超过 5000 元需附合同或协议】
- 语气：正式但不强硬
```

---

## 场景六：整理预算管理规范说明

**模板：**
```
请帮我将以下预算管理要求整理成一份清晰易读的说明文档，方便各部门负责人理解和执行：

要求内容：
【把你整理的规则或政策点粘贴进来，哪怕是零散的也没关系】

请整理成：
- 适用范围
- 核心规则（分条列出）
- 常见问题 Q&A（3-5 条）
- 联系人信息占位
```

## 财务场景的 AI 使用边界

| ✅ AI 擅长帮你做的 | ❌ AI 不能替代你做的 |
|----------------|----------------|
| 写文字性分析和说明 | 财务数字的核算和验证 |
| 解释政策条款 | 会计处理的最终判断 |
| 生成汇报框架 | 敏感财务数据的处理 |
| 起草邮件和通知 | 税务申报和合规决策 |
| 整理管理规范文档 | 审计底稿和法定报告 |
